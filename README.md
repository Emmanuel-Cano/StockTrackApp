## **Descripcción**
Esta aplicación es un gestor de inventario bien perrón, diseñado para que cualquier usuario o cualquier empresa que cuente con almacen pueda llevar el control de sus productos de manera rápida, sencilla y sin complicaciones.
Primero podrás registrarte y acceder a tu cuenta, asegurando que tu información quede guardada y protegida.

Una vez dentro, podrás añadir productos junto con todas sus características importantes:

Nombre del producto

Descripción

Cantidad

Fecha de registro

Tipo

Imagen del producto para identificarlo de volada

En pocas palabras: una app hecha para llevar tu inventario sin fallas, sin enredos y con todo el flow que necesita cualquier administrador de inventario.
